Whitelist the $LDAT transfer relay

Owner-only. Signs one transaction: setDistributor(relay, true) on the live $LDAT proxy. This activates wallet-to-wallet transfers (via the relay, 1% burn fee). Reversible later with the same call set to false. Lineascan "Write as Proxy" is broken for this Solady clone, so use this page. No libraries, no network calls except your wallet's RPC.
